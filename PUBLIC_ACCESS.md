# Публикация сайта в интернет

Этот режим запускает сайт на всех сетевых интерфейсах компьютера:

```powershell
.\Start-PublicHttp.ps1
```

После запуска сайт будет слушать порт `5028`.

## Что нужно сделать на компьютере

Открой PowerShell от имени администратора и разреши входящие подключения:

```powershell
New-NetFirewallRule -DisplayName "Class Engineering Support 5028" -Direction Inbound -Protocol TCP -LocalPort 5028 -Action Allow
```

## Что нужно сделать на роутере

В настройках роутера добавь проброс порта:

```text
External port: 5028
Internal IP: локальный IP этого компьютера
Internal port: 5028
Protocol: TCP
```

Локальный IP компьютера можно посмотреть командой:

```powershell
ipconfig
```

После этого сайт должен открываться извне по адресу:

```text
http://ТВОЙ_СТАТИЧЕСКИЙ_IP:5028
```

## Важно про безопасность

HTTP передает логины и пароли без шифрования. Для реальной работы через интернет лучше подключить домен и HTTPS, например через Caddy, Nginx или IIS reverse proxy. Тогда наружу открываются порты `80` и `443`, а приложение остается на локальном `5028`.
