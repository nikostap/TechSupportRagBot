# История версий

## 1.1.2 — 20.07.2026

- QA с пометкой «Не указан» в поле серийного номера теперь считается QA без номера и применяется ко всем станкам выбранной модели.
- Статистика API использует фактическое списание AITUNNEL из ответа API и учитывает минимальную стоимость embeddings — 0,01 ₽.

## 1.1.1 — 20.07.2026

- Автообновление сервера теперь сверяет commit развёрнутого приложения с `origin/main` и пересобирает контейнер при рассинхронизации.

## 1.1.0 — 20.07.2026

- QA без модели и серийного номера применяется ко всем станкам.
- QA только с моделью применяется ко всем серийным номерам этой модели и совместимым сериям моделей.
- Добавлены сортировка QA без перезагрузки, выбор записей и массовая верификация/удаление.
- Добавлена статистика API-запросов и расходов по назначению и моделям.
- Добавлена круговая диаграмма расходов и подпись версии внизу сайта.

## 1.0.0

- Базовая версия Class-Engineering Support: обращения, RAG, QA, оборудование, пользователи и учёт времени.
